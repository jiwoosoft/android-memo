## 🚀 v2.0.1+43 (2025-07-06) - MAJOR 업데이트

### 🆕 새로운 기능
- **🔐 지문인증 시스템 추가** - PIN 번호와 지문인증 중 선택 가능
- **🎯 통합 인증 관리** - AuthService를 통한 체계적인 인증 시스템
- **⚙️ 인증 방법 설정** - 설정에서 PIN/지문인증 방법 변경 가능
- **🔄 자동 생체인증** - 앱 재시작 시 자동으로 생체인증 실행

### 🔧 개선사항
- **🏗️ 인증 아키텍처 개선** - 기존 PIN 시스템을 AuthService로 통합
- **🎨 새로운 인증 UI** - 직관적인 설정 화면과 로그인 화면
- **🔒 보안 강화** - Flutter Secure Storage를 활용한 안전한 데이터 저장
- **📱 Android 권한 추가** - 생체인증 사용을 위한 필수 권한 설정

### 🐛 버그 수정
- **🔧 세션 관리 개선** - 로그인 후 세션 PIN 설정 로직 개선
- **🎯 라우팅 최적화** - 새로운 인증 화면들과의 원활한 네비게이션

### 📱 기술적 변경사항
- **버전**: v2.0.1 (Major 업데이트)
- **빌드 번호**: 43
- **새 의존성**: local_auth ^2.1.8, flutter_secure_storage ^9.0.0
- **Android 권한**: USE_BIOMETRIC, USE_FINGERPRINT 추가
- **최소 Android 버전**: API 21+ (Android 5.0)
