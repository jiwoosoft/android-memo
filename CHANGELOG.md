## 📦 v2.1.2 - 2024-12-28

### 🐛 버그 수정
- **👆 지문인증 실패 문제 완전 해결** - `biometricOnly: false`로 변경하여 호환성 대폭 향상
- **🔧 인증 옵션 최적화** - `stickyAuth: false`, `useErrorDialogs: true` 등 안정성 개선
- **📊 상세한 오류 진단** - PlatformException 처리로 정확한 실패 원인 파악 가능
- **💬 구체적인 오류 메시지** - 각 실패 케이스별 맞춤형 해결 방법 제공

### 🔧 개선사항
- **🎯 호환성 향상** - 더 많은 Android 기기에서 지문인증 정상 작동
- **🔍 실시간 오류 분석** - 13가지 오류 코드별 상세한 진단 및 안내
- **💡 사용자 가이드 강화** - 지문 센서 청소, 손가락 상태 등 실용적인 해결책 제공
- **🎉 성공 피드백 추가** - 지문인증 성공 시 명확한 성공 메시지 표시

### 📱 기술적 변경사항
- **버전**: v2.1.2+68 (Patch 업데이트)
- **인증 옵션 변경**: 
  - `biometricOnly: false` (PIN 폴백 허용)
  - `stickyAuth: false` (더 유연한 인증)
  - `useErrorDialogs: true` (시스템 오류 표시)
  - `sensitiveTransaction: true` (보안 강화)
- **예외 처리 강화**: PlatformException 13가지 케이스 처리
- **디버깅 개선**: 모든 지문인증 과정 상세 로깅

### 🎯 해결된 오류 코드
- **NotAvailable**: 생체인증 사용 불가
- **NotEnrolled**: 등록된 지문 없음
- **LockedOut**: 일시적 잠금
- **PermanentlyLockedOut**: 영구 잠금
- **BiometricOnlyNotSupported**: 전용 모드 미지원
- **UserCancel**: 사용자 취소
- **UserFallback**: PIN 선택
- **SystemCancel**: 시스템 취소
- **InvalidContext**: 잘못된 컨텍스트
- **BiometricNotRecognized**: 인식 실패
- **기타**: 예상치 못한 오류

### 🎯 사용자 가이드
**지문인증이 실패하는 경우:**
1. **지문 센서 청소**: 부드러운 천으로 센서를 깨끗이 닦기
2. **손가락 상태 확인**: 깨끗하고 건조한 상태 유지
3. **천천히 터치**: 센서에 지문을 천천히 대기
4. **다른 지문 시도**: 등록된 다른 손가락 사용
5. **지문 재등록**: 기기 설정에서 지문 다시 등록

- **SharedPreferences + SecureStorage 이중 저장**: 해시된 PIN과 원본 PIN을 각각 저장하여 복구 가능
- **저장 즉시 검증 시스템**: PIN 저장 후 바로 검증하여 문제 조기 발견
- **2단계 검증 로직**: 1차 해시 비교, 2차 원본 비교로 이중 안전장치

- **UTF-8 인코딩 명시**: `allowMalformed: false` 옵션으로 인코딩 일관성 보장
- **자동 복구 시스템**: 해시 저장이 깨진 경우 SecureStorage에서 자동 복구
- **상세 단계별 로깅**: 저장과 검증의 모든 과정을 추적 가능
- **완전한 초기화**: 모든 저장소의 PIN 관련 데이터 완전 삭제

### 🐛 버그 수정
- **PIN 설정 후 로그인 실패 문제**: 해시 불일치로 인한 무한 PIN 입력 화면 반복 해결
- **인코딩 불일치 문제**: UTF-8 인코딩 명시로 문자열 변환 일관성 확보
